# Trust and Safety

## Core Rule

Approval is the only path that mutates ledger truth.

## Practical Meaning

- order-like evidence can create pending invoices
- payment-like evidence can create payment proofs
- imported files can create structured evidence
- none of those paths can mark an invoice paid on their own

## Why This Matters

Collections tools fail when they create false confidence. Sah.Bukti is designed to keep evidence useful without letting evidence become accounting truth by accident.

## Public Repo Rules

- no committed secrets
- no live owner numbers
- no public QR linking controls
- no internal planning scraps presented as product documentation
